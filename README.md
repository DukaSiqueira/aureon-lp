# Barbearia - Landing Page

Landing page premium e elegante para barbearia criada com Next.js, TypeScript e Tailwind CSS.

## Características

- Design responsivo, moderno e minimalista
- Identidade visual premium com sensação de marca de alta qualidade
- Paleta de cores sofisticada:
  - Azuis escuros: #00183A, #001126, #000D1C (cores principais)
  - Cinzas: #DDDDDB, #ACADB5, #9595A0, #676977 (fundos claros e elementos de apoio)
- Tipografia limpa e legível com fonte Inter
- Espaço visual otimizado para fotos e vídeos

### Animações Inovadoras

- **Scroll Animations**: Elementos aparecem suavemente ao rolar a página
- **Hover Cards 3D**: Cards com efeito tilt 3D ao passar o mouse
- **Texto Animado**: Palavras aparecem com efeito stagger
- **Background Animado**: Partículas flutuantes e gradientes em movimento
- **Parallax**: Efeitos de profundidade em seções específicas
- **Micro-interações**: Botões com animações de hover e tap
- **Shimmer Effects**: Efeito de brilho sutil em cards
- **Floating Elements**: Elementos flutuantes com movimento suave
- **Gradient Animations**: Gradientes que se movem e mudam
- **Logo com rotação**: Entrada animada do logo com rotação

### Seções

  - Hero com antes/depois
  - Agendamento rápido
  - Galeria de cortes
  - Localização das unidades
  - Feedbacks de clientes
  - Trabalhe conosco
  - Redes sociais

## Tecnologias

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion (animações)

## Como executar

1. Instale as dependências:
```bash
npm install
```

2. Execute o servidor de desenvolvimento:
```bash
npm run dev
```

3. Abra [http://localhost:3000](http://localhost:3000) no navegador

## Scripts disponíveis

- `npm run dev` - Inicia o servidor de desenvolvimento
- `npm run build` - Cria a build de produção
- `npm start` - Inicia o servidor de produção
- `npm run lint` - Executa o linter

## Customização

Para personalizar as cores, edite o arquivo `tailwind.config.ts`.

Para modificar o conteúdo, edite o arquivo `app/page.tsx`.
